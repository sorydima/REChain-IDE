# Schemas

This directory contains canonical JSON payload examples used across services.

## Files
- schemas/task_spec.json: Task submission payload.
- schemas/task_status.json: Task lifecycle status payload.
- schemas/model_result.json: Raw model output payload.
- schemas/merge_result.json: Merge decision payload.
- schemas/artifact.json: Artifact metadata payload.

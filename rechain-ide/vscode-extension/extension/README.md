# REChain VS Code Extension

## Build
From this directory:

```powershell
npm install
npm run compile
```

## Run
- Open this repo in VS Code.
- Use the command palette and run `REChain: Submit Task`.

## Configuration
- Orchestrator URL prompt defaults to http://localhost:8081

## Notes
- Diff application uses `git apply` and requires a Git workspace.
- You can preview the diff before applying.

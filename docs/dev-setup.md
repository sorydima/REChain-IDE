# Dev Setup

## Requirements
- Go 1.21+
- Node.js 18+ (for VS Code extension build)
- Git

## Setup
1. Install Go and Node.js.
2. Run `./scripts/check-go.ps1`.
3. Start services: `./scripts/dev.ps1`.

## VS Code extension
1. `cd rechain-ide/vscode-extension/extension`
2. `npm install`
3. `npm run compile`

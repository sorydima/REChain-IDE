# Terminology

Strict definitions and abbreviations.

- AI Kernel: Runtime layer that manages model drivers and policies.
- Cross-AI Mesh: Multi-model routing and aggregation fabric.
- Driver: Adapter that exposes a model as a uniform interface.
- Execution Host: A runtime environment (local, edge, or cloud).
- RAG: Retrieval-augmented generation index and retrieval system.
- Schema Version: Semantic version string for payload compatibility.

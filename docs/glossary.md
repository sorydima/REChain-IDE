# Glossary

- Agent: An autonomous task executor with a specific role.
- Artifact: Output from a task (diff, test logs, reports).
- Kernel: Runtime that manages model drivers, sandboxing, and policies.
- Model Driver: Adapter for a model backend (local or hosted).
- Orchestrator: Routes tasks to models and merges results.
- RAG: Retrieval-augmented generation index and retriever.
- Schema: Canonical JSON payloads for inter-service communication.
- Spatial Graph: 3D representation of code and dependencies.
